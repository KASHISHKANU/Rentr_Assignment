# Contractor Job & Work Order Management Module

## Tech Stack
- Backend: Java, Spring Boot, JPA, PostgreSQL
- Frontend: Flutter
- API Docs: Swagger

## User Roles
1. Agent
2. Contractor

## Workflow
1. Agent creates job
2. Contractor applies
3. Agent approves & assigns
4. Contractor submits work plan
5. Contractor raises invoice

## How to Run Backend
```bash
cd backend
mvn spring-boot:run
